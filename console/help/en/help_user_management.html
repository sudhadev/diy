Add New User
  • Move the mouse pointer into the "Users" tab in the menu bar in Admin Console Window.
  • Click on "Add Users". "Add New User" window will appear. Fill all following fields and click on the "Add" button.
    1. First Name
    2. Last Name
    3. Username
    4. Email Address
    5. Password (Here you have to give a password with at least six characters)
    6. Verify Password
 User List
  • Move the mouse pointer into the "Users" tab in the menu bar in Admin Console Window.
  • Click on "User List" tab.  Following columns are available in the User List.
    1. Row Number
    2. First Names of the Users
    3. Last Names of the Users
    4. Usernames of the Users
    5. Email Addresses of the Users
    6. Date and Time of the last visit by User
    7. Edit and Delete icons

 Change My Password
  • Move the mouse pointer into the "Users" tab in the menu bar in Admin Console Window.
  • Click on "Change My Password ". If you are still logged user, you can change your password. Fill all the fields and click on the "Change" button.
