Global Configuration
  • Move the mouse pointer into the "Global Configuration" tab in the menu bar in Admin Console Window.
  • Select an Item from the left side bar. You can give values for following sections.
    1. Emails
    2. Subscriptions
    3. Search
    4. Titles
    5. Discount
    6. Records In List
    7. Classified Ads
    8. Date and Time
    9. Registration Pending Approval Order
  • After that corresponding configuration window will appear. You can edit required fields in there.
