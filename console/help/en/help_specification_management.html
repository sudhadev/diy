Add New Specification
  • Move the mouse pointer into the "Specifications" tab in the menu bar in Admin Console Window.
  • Click on "Add Specifications". "Browse Categories" tree will display in the left side. You can add Specifications under subcategories (under third level).
  • Select a subcategory from "Browse Categories" tree in the left side.

    • (If there is no subcategory for the selected category, you have to add a new subcategory, before add Specifications by clicking on the "Add Category" button in there.)

  • If there are subcategories in the selected category, "Add New Specification" form will appear. Fill following fields and click on the "Add" button. Specification and Average Price are the required fields.
    1. Specification Name
    2. Average Price
    3. Description
    4. Keywords
 Specification List
  • Move the mouse pointer into the "Specifications" tab in the menu bar in Admin Console Window.
  • Click on "Specification List" tab. "Browse Categories" tree will display in the left side. You can list down Specifications under each subcategory (third level) by clicking on it.

    • (If there is no subcategory for the selected category, you can add a new subcategory by clicking on the "Add Category" button in there. Or otherwise if there are no Specifications in the selected subcategory, you can add a new Specification by clicking on the "Add Specification" button in there.)

  • If there are specifications in the selected category, Specification List will appear. Following columns are available in the Specification List.
    1. Row Number
    2. Name of the Specification
    3. Description of the Specification
    4. Average Price of the Specification
    5. Edit and Delete icons
