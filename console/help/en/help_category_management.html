Add New Category
  • Move the mouse pointer into the "Categories" tab in the menu bar in Admin Console Window.
  • Click on "Add Categories". "Browse Categories" tree will display in the left side. You can add Categories under under first and second levels.
  • Select a subcategory from "Browse Categories" tree in the left side.
  • Fill following fields and click on the "Add" button.
    1. Category Name
    2. Parent Category
    3. Image (This field is display only for second level.)
    4. Category Description
    5. Category Status
 Category List
  • Move the mouse pointer into the "Categories" tab in the menu bar in Admin Console Window.
  • Click on "Category List" tab. "Browse Categories" tree will display in the left side. You can list down subcategories by clicking on a category.
  • If there are subcategories in the selected category, Category List will appear. Following columns are available in the Category List.
    1. Row Number
    2. Name of the Category (There is an image with zoom icon for corresponding filed. By clicking on the , you can see the enlarge image. This will appear only for third level categories.)
